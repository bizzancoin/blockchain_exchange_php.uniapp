## 二、字段阅读约定

| 字段模式 | 含义 |
| --- | --- |
| id | 通常为主键ID。 |
| *_id | 通常为关联ID，例如 user_id、currency_id、order_id。 |
| created_time / updated_time | 多为 Unix 时间戳字段。 |
| status / type / is_* | 多为状态、类型或布尔开关，需要业务枚举说明。 |
| decimal(20,8) | 常用于资产数量、价格、手续费等高精度数值。 |
| password / token / google | 属于敏感安全字段，对外分享时建议隐藏或仅说明字段用途。 |
